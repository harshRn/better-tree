# better-tree

v 0.0.1 not-stable

A better version of linux `tree` command.

---

## How to try?

1. `git clone https://github.com/tesla33io/better-tree.git ~/better-tree`
2. `cd ~/better-tree`
3. `cargo build`
4. `cp ./target/debug/tree-rust ./`

Now you can use `tree-rust` binary to execute the program.
